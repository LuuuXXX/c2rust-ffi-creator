## 模块：`<模块名>` 北向接口（对外暴露）

### `function_name` [必填]

**签名：** [必填]
```c
return_type function_name(type1 param1, type2 param2);
```

**功能描述：** [必填]
> 一句话说明函数的功能。

**参数：** [必填]

| 参数名 | 类型 | 是否可 NULL | 取值范围 | 含义 |
|--------|------|------------|----------|------|
| `param1` | `type1` | 否 | [0, INT_MAX] | 说明 |
| `param2` | `type2` | 是（表示默认值） | 任意 | 说明 |

**返回值：** [必填]

| 值 | 含义 |
|----|------|
| `0` | 成功 |
| `-EINVAL` | 参数无效（param1 为负数） |
| `-ENOMEM` | 内存分配失败 |
| `NULL` | 未找到 / 错误 |

**内存所有权：** [必填]

- 参数指针：调用方拥有，不会被函数持有（或：函数会持有，调用方不能释放）
- 返回指针：调用方负责释放（使用 `xxx_free()` 释放）/ 静态内存，不需释放 / 函数内部管理

**线程安全：** [必填]

- [x] 线程安全（无全局状态，纯函数）
- [ ] 需要外部同步（修改共享状态 `g_xxx`）
- [ ] 不可在多线程中使用

**副作用：**

- 无 / 修改全局配置 / 写入文件 / 网络请求 / 启动线程

**错误处理：**

- 若返回错误，`errno` 是否被设置：是 / 否
- 具体错误码含义：（见 `include/errors.h`）

## 填写示例：`calc` 模块

### `calc_add`

**签名：**
```c
int calc_add(int a, int b, int *result);
```

**功能描述：**
计算两个整数之和，结果写入 `*result`。

**参数：**

| 参数名 | 类型 | 是否可 NULL | 取值范围 | 含义 |
|--------|------|------------|----------|------|
| `a` | `int` | N/A | `[INT_MIN, INT_MAX]` | 被加数 |
| `b` | `int` | N/A | `[INT_MIN, INT_MAX]` | 加数 |
| `result` | `int *` | 否 | N/A | 输出参数，存储计算结果 |

**返回值：**

| 值 | 含义 |
|----|------|
| `0` | 成功，`*result` 已写入 |
| `-EINVAL` | `result` 为 NULL |
| `-EOVERFLOW` | 整数溢出 |

**内存所有权：**
- `result`：调用方提供并拥有，函数只写入值，不持有指针

**线程安全：**
- [x] 线程安全（无全局状态，仅操作参数）

**副作用：**
- 无

**错误处理：**
- 溢出时设置 `errno = EOVERFLOW`
